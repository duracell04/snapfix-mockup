# Welcome to your https://akalabs.io/ project

## Project info

**URL**: https://akalabs.io/

## Stack

- Next.js (App Router, TypeScript)
- Tailwind CSS + shadcn/ui
- pnpm (package manager)

## Local development

```sh
pnpm install
pnpm dev
```

Build for production:

```sh
pnpm build
pnpm start
```
